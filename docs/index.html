Yuxu's Game of Life

A GPU-Accelerated Artificial Life Simulation with Emergent Neural Evolution — extending Conway's Game of Life into a massively parallel neuroevolution ecosystem.

View on GitHub Watch Demo Read Blog Post

Real-time simulation with 10,000+ neural agents competing, reproducing, and evolving

Key Finding

A trained "super-lineage" starting from just 1% of the population achieved 100% ecosystem dominance within ~5000 generations, demonstrating the compound advantage of hybrid RL-neuroevolution architectures.

Key Features

⚡

Massive GPU Parallelization

10,000+ simultaneous neural agents evaluated in parallel via PyTorch. Vectorized operations using torch.einsum for batched forward passes with pre-allocated tensor buffers.

🧠

Hybrid Evolution Architecture

Combines within-lifetime reinforcement learning (policy gradient) with cross-generational neuroevolution (sexual reproduction with crossover and mutation).

🧬

12-Dimensional Genomic Identity

8D neural fingerprint (genotype) + 4D chemical affinity (phenotype). Cosine similarity-based compatibility enables emergent speciation.

🌿

Emergent Ecological Dynamics

No predefined species — clusters emerge naturally from genome similarity. 4-chemical signaling system with diffusion and decay creates rich ecological interactions.

Terminal
